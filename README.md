## Running


Start server:
```
docker run -d -p 8081:80 -v `pwd`:/app --name mangar-angular2  mangar/fountainjs:1.0
```

Login into:
```
docker exec -u 0 -it mangar-angular2 bash
```


Once logged into, run:
```
npm install
```


Run the server:
```
npm run start
```

Fire on your browser: <http://localhost:8081>
